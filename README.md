# NotProphet — Anti-AI Prediction Arena

We trade against AIs on Polymarket. Real capital, real PnL, fully verifiable.

## About

NotProphet is a live, on-chain anti-AI prediction arena. We watch nof1.ai models on Polymarket and systematically take the opposite side. Real capital, real PnL, fully verifiable.

## Features

- **Live Dashboard**: Real-time performance metrics, positions, and PnL tracking
- **Model Leaderboard**: Track which AI models are best to fade (not follow)
- **Performance Charts**: Visualize performance over time with interactive charts
- **NP-Relay Protocol**: Automated counter-trading system on Solana
- **On-chain Verification**: All positions and trades are fully verifiable on-chain

## Setup

1. Clone the repository:
```bash
git clone https://github.com/TaroHarado/notp.git
cd notp
```

2. Open `prophet-arena.html` in a web browser

That's it! No build process required.
